To run the system

If your desktop doesn't have XAMPP.
Follow these steps:
1.	Install XAMPP (https://www.apachefriends.org/download.html)
2.	Open XAMPP Control Panel (run as administrator)
3.	Start Apache
4.	Start MySQL
5.	When Apache and MySQL is running click MySQL Admin (http://localhost/phpmyadmin/)
6.	Create new database (click New)
7.	Name it "ojt_project" then Create
8.	Import ojt_project.sql file
9.	If successful congrats!

To run the system on the desktop
Open XAMPP Apache and MySQL
Run File Monitoring.exe
